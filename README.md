# Student Grade Calculator

## Description
This Python program calculates a student's grade based on the marks entered by the user.

## Features
- Takes student name as input
- Accepts marks from 0 to 100
- Calculates grade automatically
- Displays a motivational message
- Validates invalid marks

## Technologies Used
- Python 3

## How to Run

```bash
python grade_calculator.py
```
